## Endpoint :

### List of available endpoints:

-   POST /users
-   POST /authentication
-   POST /tweets
-   GET /tweets
-   PATCH /tweets/:tweet_id
-   POST /tweets/:tweet_id/comments
-   GET /tweets/:tweet_id/comments
-   PATCH /tweets/:tweet_id/comments/:comment_id
-   DELETE /tweets/:tweet_id/comments/:comment_id
-   DELETE /tweets/:tweet_id


## 1. POST /users

### Description

-   register new user

### request

```json
 {
  "username": "bejo",
  "password": "12345"
}
```


### Response

Response (201)

```json
{
  "status": "Success",
  "message": "Success",
  "data": {
    "id": 1,
    "username": "bejo",
    "createdAt": "2023-11-19T21:20:41.000Z",
    "updatedAt": "2023-11-19T21:20:41.000Z"
  }
}
```

### validation

### Description

-  username is required 

### request

```json
 {
  "username": "",
  "password": "12345"
}
```


### Response

Response (400)

```json
{
  "name": "BadRequest",
  "message": "Username is required.",
  "code": 400,
  "className": "bad-request"
}
```

-  username is required 

### request

```json
  "username": "bejo",
  "password": ""
}
```
